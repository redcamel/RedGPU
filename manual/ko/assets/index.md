# 확장 객체 (Extended Objects)

외부 모델 로딩 및 2D 요소를 활용하여 씬의 콘텐츠를 풍성하게 만드는 방법을 다룹니다.

## 학습 주제
1. **[GLTF Loader](./model-loading/index.md)**: 외부 3D 모델 파일(.gltf, .glb) 불러오기
2. **[Sprite](./sprite/index.md)**: 빌보드 아이콘 및 애니메이션 스프라이트 활용
3. **[Text Field](./text-field/index.md)**: 3D 월드 내 텍스트 배치
