# RedGPU

RedGPU는 WebGPU 기반의 차세대 3D 그래픽 엔진입니다.


고성능 렌더링 파이프라인, 유연한 카메라 시스템, 다양한 조명 및 재질, 후처리 효과 등을 제공하여 웹 환경에서 강력한 그래픽 애플리케이션을 구축할 수 있도록 돕습니다.
