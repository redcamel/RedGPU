# Picking

마우스 피킹, 레이캐스팅 및 이벤트 처리를 위한 핵심 기능을 제공합니다.


GPU 기반 픽셀 피킹 시스템과 CPU 기반 레이캐스팅(2D/3D)을 모두 지원하여 정밀한 객체 선택 및 상호작용을 가능하게 합니다.
