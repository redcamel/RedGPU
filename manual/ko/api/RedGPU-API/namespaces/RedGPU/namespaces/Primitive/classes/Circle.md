# Class: Circle

Defined in: [src/primitive/Circle.ts:20](https://github.com/redcamel/RedGPU/blob/302ea63caf5f5ecf51af7f70ca3a0940bde79a48/src/primitive/Circle.ts#L20)

Circle(원형) 기본 도형 클래스입니다.

반지름, 세그먼트 등을 기반으로 2D 원형 데이터를 생성하여 관리합니다.

### Example
```typescript
// 반지름 1, 32세그먼트 원형 생성
const circle = new RedGPU.Circle(redGPUContext, 1, 32);
```
