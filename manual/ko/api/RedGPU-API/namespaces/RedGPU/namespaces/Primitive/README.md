# Primitive

씬 구성을 위한 다양한 3D 기본 도형(Primitive)들을 제공합니다.


Box, Sphere, Plane, Torus 등 표준 기하 도형들을 손쉽게 생성하고 Mesh의 지오메트리로 사용할 수 있습니다.
