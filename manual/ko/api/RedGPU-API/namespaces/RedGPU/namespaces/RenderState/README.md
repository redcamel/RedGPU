# RenderState

렌더 파이프라인의 조작을 위한 다양한 렌더 상태 관리 클래스들을 제공합니다.


블렌딩, 깊이/스텐실 테스트, 프리미티브 토폴로지 및 컬링 등 GPU의 렌더링 동작을 세밀하게 정의하는 도구들을 포함합니다.
