# 기본 객체 (Basic Objects)

3D 공간을 구성하는 가장 핵심적인 요소들을 배우는 단계입니다. RedGPU의 세계를 시각적으로 채우고, 객체들 간의 관계를 정의하며, 사실적인 재질과 조명을 적용하는 방법을 다룹니다.

## 학습 목표

이 섹션을 마치면 다음과 같은 작업을 수행할 수 있습니다.

- **메시(Mesh) 생성**: 지오메트리와 머티리얼을 결합하여 3D 물체를 화면에 출력합니다.
- **계층 구조(Scene Graph) 설계**: 부모-자식 관계를 통해 복잡한 구조를 만들고 애니메이션을 제어합니다.
- **텍스처(Texture) 적용**: 단순한 색상을 넘어 실제 이미지를 물체 표면에 입힙니다.
- **조명(Light)과 재질**: 다양한 광원을 추가하고 빛에 반응하는 사실적인 재질을 적용합니다.
- **그림자(Shadow)**: 공간의 입체감을 완성하는 그림자 시스템을 활용합니다.

## 포함된 주제

1. **[메시 (Mesh)](./mesh.md)**: RedGPU의 가장 기본 단위인 메시와 그 구성 요소(지오메트리, 머티리얼)에 대해 배웁니다.
2. **[Scene Graph](./scene-graph.md)**: 객체 간의 계층 구조와 변형 상속의 원리를 이해합니다.
3. **[텍스처 (Texture)](./texture.md)**: 비트맵 이미지를 불러와 물체에 입히는 방법을 알아봅니다.
4. **[조명 (Light)](./light.md)**: PhongMaterial과 다양한 광원을 통해 공간에 생동감을 불어넣습니다.
5. **[그림자 (Shadow)](./shadow.md)**: 사실적인 공간감을 위한 그림자 설정법을 익힙니다.

이제 첫 번째 단계인 **메시**부터 시작해 보겠습니다.
