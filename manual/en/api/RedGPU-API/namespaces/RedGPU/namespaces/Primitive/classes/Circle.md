# Class: Circle

Defined in: [src/primitive/Circle.ts:20](https://github.com/redcamel/RedGPU/blob/ed852b773d59f6e9bad76f340c03dc4200b077ba/src/primitive/Circle.ts#L20)

Circle primitive geometry class.

Generates and manages 2D circular data based on radius, segments, etc.

### Example
```typescript
// 반지름 1, 32세그먼트 원형 생성
const circle = new RedGPU.Circle(redGPUContext, 1, 32);
```
