# Material


Provides various materials (Bitmap, Color, Phong, etc.) and related constants (BLEND_MODE, COMPOSITE_MODE, TINT_BLEND_MODE).


Through each material class and constant, detailed material production and management such as various rendering effects, material property control, and blending/composite mode settings are possible.
