Function init

WebGPU를 비동기적으로 초기화합니다. 초기화에 실패한 경우 선택적으로 제공된 콜백 함수를 호출합니다.
Asynchronously initializes WebGPU. If initialization fails, it invokes an optionally provided callback function.
