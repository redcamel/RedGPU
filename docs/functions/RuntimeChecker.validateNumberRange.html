    Function validateNumberRange

    • validateNumberRange(
          value: number,
          minRange?: number,
          maxRange?: number,
      ): boolean

      주어진 값이 지정한 범위 내의 숫자인지 검증합니다.

      value, minRange, maxRange가 모두 number 타입인지 확인하며,

      value가 minRange 이상, maxRange 이하인지 검사합니다.

      범위를 벗어나거나 타입이 맞지 않으면 예외를 발생시킵니다.

      Parameters

      • value: number

        검증할 값

      • OptionalminRange: number = -Number.MAX_VALUE

        허용되는 최소값 (기본값: -Number.MAX_VALUE)

      • OptionalmaxRange: number = Number.MAX_VALUE

        허용되는 최대값 (기본값: Number.MAX_VALUE)

      Returns boolean

      값이 범위 내의 숫자이면 true

      Throws

      값이 숫자가 아니거나 범위를 벗어나면 예외 발생
