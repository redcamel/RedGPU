    Class Camera2D

    2D 카메라 클래스입니다. x, y 위치와 모델 행렬을 관리하며, 이름 지정 및 위치 설정 기능을 제공합니다.

    Example

    const camera = new RedGPU.Camera.Camera2D();
    camera.x = 100;
    camera.y = 50;
    camera.setPosition(200, 100);
