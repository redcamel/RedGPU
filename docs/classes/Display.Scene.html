    Class Scene

    Scene 클래스는 View에서 렌더링할 장면을 구성하는 공간입니다.

    조명, 그림자, 배경색 등의 설정을 포함하며, 모든 시각적 객체의 루트 컨테이너로 작동합니다.

    View3D 또는 View2D와 연결되어 화면에 출력되는 대상이며, 시각적 콘텐츠를 배치하는 무대 역할을 합니다.

    const scene = new RedGPU.Display.Scene();
