    Class OBB

    3차원 Oriented Bounding Box(OBB) 클래스

    3차원 공간에서 임의의 방향을 갖는 경계 상자(Oriented Bounding Box)를 표현합니다.

    중심 좌표, 반치수(half extents), 방향 행렬, 반지름 등의 정보를 제공합니다.

    교차 및 포함 여부 판별, 복제 기능을 지원합니다.
