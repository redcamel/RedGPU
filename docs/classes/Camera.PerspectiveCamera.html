    Class PerspectiveCamera

    원근 투영 카메라(PerspectiveCamera) 클래스입니다. x, y, z 위치, 회전, 시야각, 클리핑, 모델 행렬, 이름 등을 관리합니다. lookAt, setPosition 등 카메라 제어 기능을 제공합니다.

    Example

    const camera = new RedGPU.Camera.PerspectiveCamera();
    camera.x = 10;
    camera.y = 5;
    camera.z = 20;
    camera.fieldOfView = 75;
    camera.lookAt(0, 0, 0);
