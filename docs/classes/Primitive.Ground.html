    Class Ground

    Ground(그라운드) 기본 도형 클래스입니다. XZ 평면에 배치된 평면 메시로, 세그먼트, UV 스케일, Y축 뒤집기 등 다양한 파라미터를 지원합니다.

    Example

    // 10x10 크기, 10x10 세그먼트, UV 2배, Y축 뒤집기 없이 생성 및 씬에 추가
    const ground = new RedGPU.Primitive.Ground(redGPUContext, 10, 10, 10, 10, 2, false);
