    Class DOF

    피사계 심도(DOF, Depth of Field) 후처리 이펙트입니다. CoC(혼란 원) 계산과 블러를 결합해 사실적인 심도 효과를 제공합니다. 다양한 사진/영상 스타일 프리셋 메서드를 지원합니다.

    Example

    const effect = new RedGPU.PostEffect.DOF(redGPUContext);
    effect.focusDistance = 10;
    effect.aperture = 2.0;
    effect.maxCoC = 30;
    effect.setCinematic(); // 시네마틱 프리셋 적용
    view.postEffectManager.addEffect(effect);
