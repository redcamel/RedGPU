    GPU 블렌딩 연산을 위한 블렌드 팩터 옵션

    소스와 대상 색상을 어떻게 혼합할지 결정하는 팩터를 정의합니다. 블렌드 방정식: result = srcFactor * srcColor op dstFactor * dstColor

    Type Declaration

      블렌드 상수 색상을 사용합니다.

      파이프라인에 설정된 상수 색상 값을 팩터로 사용합니다.

    • ReadonlyDST: "dst"

      대상 색상을 사용합니다.

    • ReadonlyDST_ALPHA: "dst-alpha"

      대상 알파 값을 사용합니다.

    • ReadonlyONE_MINUS_DST_ALPHA: "one-minus-dst-alpha"
