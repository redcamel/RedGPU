# [Feature] 디테일 인프라 구축 (Decal System)

## 📌 개요
물리 기반 공간의 사실성을 극대화하기 위해, 모델링 데이터 수정 없이 표면에 미세한 디테일을 추가할 수 있는 실시간 데칼 시스템을 구현합니다.

## 🎯 목표
- 투영(Projection) 방식의 실시간 데칼 액터 구축
- G-Buffer 및 뎁스 정보를 활용한 정밀한 표면 합성
- 오염, 균열, 탄흔 등 다양한 시각적 디테일의 자유로운 배치 지원

## 🛠️ 주요 작업 항목
- [ ] **Decal Actor 및 투영 구조 설계**
  - 박스 영역 기반의 투영 역행렬(Inverse Matrix) 연산 구현
- [ ] **Decal Shader 개발**
  - G-Buffer Normal/Albedo 레이어와의 물리적 합성 로직
  - 뎁스 정보를 활용한 정밀한 투영 무결성 확보
- [ ] **최적화 및 관리 시스템**
  - 대량의 데칼 배치 시 성능 저하 방지를 위한 드로우 콜 최적화

## 📊 기대 효과
- 웹 기반 3D 환경 특유의 '매끄러움'을 지우고 사실적인 질감 부여
- 아티스트의 씬 구성 자유도 비약적 상승
